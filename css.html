    4. HTML이란 무엇인가?

    Hypertext Markup Language (HTML) is the standard markup language for creating webpages and web applications.
    Web browers recieve HTML documents from a web server or from a web server or from local storage and render them into multimedia web pages.
    HTML describes the structure of a web page semanically and originally included cues for the appearance of the document.
